## 2026-05-23 - Native Tooltips on Icon-Only Buttons
**Learning:** We can natively use `app:tooltipText` alongside `android:contentDescription` on icon-only `MaterialButton` elements. While `android:tooltipText` works on API 26+, it is best practice in Android to use `app:tooltipText` (from AppCompat/Material) to ensure comprehensive backwards compatibility. This provides a crucial fallback mechanism for non-screen-reader users (like mouse/pointer users or long-press interactions on touch) to discover what a button does without requiring custom tooltip wrapper components. It's a low-effort, high-impact accessibility win.
**Action:** Automatically add `app:tooltipText` mapping to the same string resource as `android:contentDescription` for all icon-only buttons as a standard practice.

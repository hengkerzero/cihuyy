## 2026-05-12 - Use TextInputLayout for Form Usability
**Learning:** Replacing a standard `EditText` with Material `TextInputLayout` and `TextInputEditText` provides floating labels, clear focus states, and handles single-line inputs more cleanly. It's a noticeable UX improvement with minimal code changes.
**Action:** When I encounter raw `EditText` widgets, I should wrap them in `TextInputLayout` elements to improve form accessibility and visual feedback.

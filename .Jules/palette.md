## 2026-06-10 - Adding Tooltips to Icon-Only Buttons
**Learning:** In standard Android layouts using Material components, `android:contentDescription` is necessary for screen readers, but it doesn't automatically show visual tooltips for sighted users on long-press or mouse hover. Relying solely on `android:contentDescription` on icon-only buttons leaves sighted users guessing the action. Using `android:tooltipText` explicitly alongside it provides crucial visual feedback. Also, using `app:tooltipText` is only valid for menu items; regular views require `android:tooltipText` (requiring minSdk 26+, which matches the app's minSdk 27).
**Action:** Always pair `android:contentDescription` with `android:tooltipText` mirroring the same string resource for icon-only buttons in XML layouts to ensure both visual and programmatic accessibility.
